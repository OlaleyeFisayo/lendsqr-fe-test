# Lendsqr FE Test

This is a React, TypeScript, and Vite application for the Lendsqr frontend test.

## Documentation

Project setup and workflow details are documented in the `docs` folder:

- [Installation](docs/installation.md): install dependencies, start the dev server, build, and preview the app with pnpm.
- [Linting](docs/linting.md): explains the ESLint setup, ignored files, key rules, and lint commands.
- [Icons](docs/icons.md): explains Iconify usage and the default Tabler Icons collection.

## Quick Start

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Build the app:

```bash
pnpm build
```

This project is configured to use pnpm as its package manager.
